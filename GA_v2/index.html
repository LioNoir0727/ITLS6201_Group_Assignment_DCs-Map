Sydney Grocery DC Location Planner

This interactive map supports the selection of two distribution centres for a grocery chain serving the Sydney metropolitan area.

Map Setup

  • Initial view centred on Greater Sydney.
  • CartoDB Positron basemap for a clean presentation style.
  • Boundary, corridor, and distribution centre layers can be added next.

Selection Logic

The two recommended DC locations are Moorebank / Prestons and Eastern Creek / Erskine Park. Moorebank / Prestons is selected because it has strong access to Port Botany, intermodal freight rail, and the M5/M7 motorway network. This makes it suitable for inbound containers, refrigerated imports, and southern/south-western store replenishment. Eastern Creek / Erskine Park is selected because it is close to the M4/M7 motorway junction and is already used by major supermarket chains for chilled and fresh distribution. This makes it suitable for western, north-western and northern Sydney store coverage.

Boundary and Data Note

GCCSA, SA3 and SA2 boundary layers are loaded from local GeoJSON files converted from shapefiles. The Greater Sydney GCCSA boundary defines the metropolitan analysis area. SA3 and SA2 layers provide regional detail for presentation and future store mapping.

The LGA boundary layer is based on the Metropolitan Boundaries reference map. GCCSA is used as the official metropolitan statistical boundary, while LGA boundaries provide an easier presentation view.

Prototype Note

DC coordinates, freight corridors, and benchmark locations are approximate and intended for presentation-level spatial testing. They can be replaced with verified coordinates and official freight network datasets for final analysis.
