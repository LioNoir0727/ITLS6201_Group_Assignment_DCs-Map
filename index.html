Sydney Grocery DC Location Planner

This interactive map supports the selection of two distribution centres for a grocery chain serving the Sydney metropolitan area.

Map notes

This map supports DC location analysis for a Sydney metropolitan grocery network.

Candidate DCs are assessed using store coverage, road access, inbound freight access, land suitability, EV readiness and real-world validation.

The Sydney metropolitan boundary is the main analysis area. Greater Sydney is shown only as a wider context boundary.

SA2 population density is used as a demand proxy. Major roads are used to assess logistics accessibility.

This is a planning prototype, not an engineering or operational GIS model.
